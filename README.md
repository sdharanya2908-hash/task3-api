# Task 3 - Books REST API (Node.js + Express)

This project is part of the Web Development Internship Task 3.

## Endpoints
- GET /books → Get all books
- POST /books → Add a book
- PUT /books/:id → Update a book
- DELETE /books/:id → Delete a book

## How to Run
1. npm install
2. node server.js
3. Open http://localhost:3000/books in browser
